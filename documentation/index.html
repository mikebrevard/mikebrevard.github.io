Structure

Pawiwahan has a well-commented to easily understand.

  • < head > area which contains the meta tags and load a bunch of css files
  • < body > the whole body content such as < header >, < section > & < footer >
  • < header > area which contains menu and logo
  • < section > each section will contain main content for website
  • < footer > end of website which contains social links and end-quote/copyrights

How to configuring Google Maps Coordinates

          
Don't forget to give map of your ceremonial location

          
index.html

          
You can change telephone number, address street line and email. Just leave the rest

          

          
Make sure you have < div id="map" >< /div > this is a placeholder to render the map, also you can set your latitude, longitude & title marker through data attributes data-lat="-8.1037781", data-long="115.0890078" and data-title="My wedding here!" match yours.

          
Also since google update to v.3.0 you need to make your own API to put at bottom of page

          
< script async defer src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_HERE&callback=initMap" >< /script >
